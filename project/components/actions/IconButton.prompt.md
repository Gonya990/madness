A single-icon button for toolbars, editor chrome, and dense controls. Always pass `label` for accessibility.

```jsx
<IconButton icon="play" label="Run" variant="primary" />
<IconButton icon="search" label="Search" />
<IconButton icon="more-vertical" label="More" variant="subtle" />
<IconButton icon="user" label="Account" round variant="outline" />
```

- Variants: `ghost` (default), `subtle`, `outline`, `primary`.
- Sizes `sm | md | lg`; `round` for avatar-style controls.
