# UniCode — Design System

Витрина дизайн-системы **UniCode**: браузерная платформа для совместной разработки
(IDE в браузере + playground + обучение). Собрана на Vite + React + TypeScript.
Весь интерфейс на русском, тёплая коралловая палитра, светлая и тёмная темы.

## Возможности

- **25 компонентов** — кнопки, формы, фидбек, данные, навигация, код, оверлеи, бренд, иконки
- **5 экранов** — редактор (IDE), дашборд, мобильное приложение (iOS), маркетинг, документация
- **Токены** — цвета/типографика/отступы/тени/анимации через CSS-переменные
- **Тёмная тема** — переключатель на главной, значение запоминается, учитывается системная тема
- **Иконки** — статический реестр только используемых иконок `lucide-react` (лёгкий бандл)

## Быстрый старт

```bash
npm install
npm run dev      # http://localhost:5173
```

Прод-сборка и предпросмотр:

```bash
npm run build    # tsc -b && vite build → dist/
npm run preview
```

## Структура

```
src/
  components/     UI-компоненты по группам (actions, forms, feedback, …)
  pages/          5 экранов витрины + Home-лаунчер
  styles/tokens/  CSS-токены дизайн-системы
  lib/            ui-хелперы (injectStyle, cx) и управление темой
  App.tsx         маршрутизация (react-router), ленивые чанки экранов
  main.tsx        точка входа
project/          исходная дизайн-система (прототипы и документация)
DESIGN.md         полный дизайн-гайд (визуальные основы, компоненты)
```

## Стек

Vite 5 · React 18 · TypeScript 5 · React Router 6 · lucide-react
